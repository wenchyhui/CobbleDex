# Contribuindo para CobbleDex

Obrigado por querer contribuir! Pequeno guia:

1. Fork no GitHub
2. Crie uma branch: `git checkout -b feat/minha-idea`
3. Faça mudanças pequenas e com commits claros
4. Rode os testes `pytest`
5. Abra um Pull Request descrevendo a mudança

Marque issues com `good first issue` para começar. Sinta-se livre para abrir uma issue antes de trabalhar.

Respeite o `CODE_OF_CONDUCT.md`.
